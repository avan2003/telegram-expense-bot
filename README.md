# Telegram Expense Bot

Private Telegram bot to track expenses and send monthly summaries.

## Features
- Log expenses by sending messages like `12.50 lunch`
- `/today` to see today's spending
- `/month` to see current month summary
- Monthly summary script for automatic Telegram message

## Environment variables
- `BOT_TOKEN`
- `ALLOWED_USER_ID`
- `DB_PATH`
